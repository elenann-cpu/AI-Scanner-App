AI Scanner is a modern Android application designed for Optical Character Recognition (OCR) with a focus on data security and user privacy. The app allows users to quickly digitize printed text while protecting sensitive information through intelligent data masking.

✨ Key Features
AI-Powered OCR: High-precision text recognition using advanced machine learning models.

Data Masking & Privacy: Automatic detection and masking of sensitive data, such as email addresses and credit card numbers, to ensure information security.

Modern UI/UX: Intuitive user interface featuring a polished design with a professional "Ink Blue" theme for better usability.

Quick Actions: Seamlessly copy scanned text to the clipboard or clear the screen with a single tap.

🛠 Tech Stack
Language: Kotlin 

Platform: Android SDK

Environment: Android Studio

Version Control: Git & GitHub

🚀 How It Works
Scan: Use the "Scan" feature to recognize text from any document or image.

Process: The recognized text is displayed on the main screen, with sensitive information automatically masked.

Copy: Use the "Copy" action to save the processed text to your clipboard for use in other applications.

🔒 Security & Integrity
Built with a commitment to professional integrity, this project explores the practical implementation of data privacy mechanisms, ensuring that digitization does not compromise personal confidentiality.
